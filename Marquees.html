MARQUEE TAGS

  • It is basically responsible for the movement of the text in the web pages.
  • Using this we can move heading,paragraphs and images also.
  • Basically we are having three type of movement.
      • 1.Scroll
    • It is default movement of the elements in the web pages.
    • In this the default direction of the movement is the right to left.

      • Syntax: < marquee behavior="" direction="" >HEADING< /marquee >

      HEADING

       2.Alternate
    • For this we need to use the behavior="" attribute of the marquee tag to change the movement of the elements.
    • In this the default direction of the movement is the right to left.

      • Syntax: < marquee behavior="Alternate" direction="" >HEADING< /marquee >

      HEADING
    • For changing the direction we need to use direction="" attribute of marquee tag.

        • We are having total four type of direction o movement

      1. Right

        2. Syntax: < marquee behavior="Alternate" direction="right" >HEADING< /marquee >

        HEADING
      2. left

        3. Syntax: < marquee behavior="Alternate" direction="left" >HEADING< /marquee >

        HEADING
      3. Up

        4. Syntax: < marquee behavior="Alternate" direction="up" >HEADING< /marquee >

        HEADING
      4. Down

        5. Syntax: < marquee behavior="Alternate" direction="down" >HEADING< /marquee >

        HEADING
    • We can also control the speed of movement of the elements in any derection by using scrollamount="" attribute of the marquee tag.

      • Syntax: < marquee behavior="Alternate" direction="down" scrollamount="300" >HEADING< /marquee >

      HEADING

       3.Slide
    • For this we need to use the behavior="" attribute of the marquee tag to change the movement of the elements.
    • In this the default direction of the movement is the right to left.
    • We can also change the direction and speed of movement.

      • Syntax: < marquee behavior="slide" direction="" >HEADING< /marquee >

      HEADING
