FORM TAG

By using < form action="">< /form> tag we can create form in html for web pages.

The < form> tag contains action="" attribute to navigate the form page to other pages after submitting to the next page.

The < form> tag is used to create an HTML form for user input.

The < form> element can contain one or more of the following form elements :

< label>
It is used for indicate what we need to give as a input.
Sysntax : < label for="">lable< /label>

Between both label tags we can write our lable.
We can also skip for tag it is not mandatory.
< input>
It is used to take input from user.
Basically in < input> tag it contains two attributes one is type="" & name=""
In < input> tag type we need to mention what type of input we need to take .
In < input> tag name Specifies the name of a form,this helps us to store in database easily.
Type attributes in < input> tag may different elements :
  • Syntax : < input type="text" name="name"> (type="text" used for alphabet input)

    • For text type input we are havin some attributes names as required,placeholder,minlength & maxlength.
    • required is used when that data is mandatory.
    • placeholder will be display in the input area as a defalut text for the user.
    • minlength& maxlength is used to set the minimum and maximum length for any type of input except numbers.

  • Syntax : < input type="tel" name="Mobilenumber" minlength="10" maxlength="10"> (type="tel" used for telephone number input)


  • Syntax :< input type="number" min="0" max="18" name="age"> (type="number" used for integer input)

    • min & max is used to set minimum and maximum limit for integers.
  • Syntax :< input type="date" name="DOB"> (type="date" used for date input)


  • Syntax :< input type="radio" name="gender"> < input type="radio" name="gender"> (type="radio" used for choosing from multiple optoins)

    • For each choosing options we need to one input tag with radio type.

  • Syntax :< input type="file" name="resume"> (type="file" used for files submitting)


  • Syntax :< input type="email" name="email"> (type="email" used for email input)

    • This input type will accept only email format,for other input it will show error for that.
    • We can write anything related to label but it will not displayed anywhere.

  • Syntax :< input type="password" name="password" minlength="8" maxlength="16"> (type="password" used for password type input)

    • By using type as password we can store store any input type as well as we can hide password also.
    • We are also able to limit the password length by using minlength and maxlength attributes.

  • Systax :< input type="checkbox"> (type="checkbox" is used for a check box)

    • chechbox is used for putting one check box in the page for selecting the option.
< textarea>
Textarea tag is used for taking large input from users like address and feedback.
By using cols and rows attrinutes we can set and change the shape of the textarea.
  • Syntax : < textarea cols="30" rows="10" name="address">< /textarea>

< button>
It is used to put one button on web pages like submit,login and signup.
< select>
Select is used for drop down options for multiple optons.
It is always used with < option> tag.
< option>
One option tag holds one option for select drop down options.
For each and every option we need to write one option tag.
