IMAGE TAG

  • Image tag is used to insert images to the web pages.
  • We can also change the size of the images in the pages according to our need.
  • We can insert image by using < img>< /img> tag.

Attributes

  • Basically in this tag it consists of two attributes src="" and alt="".

    • Syntax : < img src="" alt="">

    1. src :-
      • src="" is used to define the source of the image.
      • We can take a source of image by coping the image source from internet and also from own device by coping the path of the file.

        • Syntax : < img src="https://wallpaperaccess.com/full/2749420.jpg" alt="">

      • We can also keep a copy of image in our project folder so we can directly access that image.

        • Syntax : < img src="koenigsegg-agera.jpg" alt="">

      • Also we can access image by using folder name from our system.

        • Syntax : < img src="folder name/img" alt="default text">

    2. alt :-

      3. It is used as a temporary text or a default text when your image will not get display due to some issues.

      Syntax : < img src="https://wallpaperaccess.com/full/2749420.jpg" alt="Natural view">

    3. height & width :-

      4. It used to adjust the height and width of the images in the web pages.

      Syntax : < img src="https://media.crictracker.com/media/attachments/1690383625521_17kohli1.jpeg" alt="Virat kohli">
