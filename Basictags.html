BASIC TAGS

  • Html tags or like keyword which is use to return html Document.
  • Each tags its own Properties which can be use for different purpose.
There are two types of tags
Single Tag < tag>
Double Tag < tag>< /tag>
  1. < !DOCTYPE>
    • The < !DOCTYPE> declaration represents the document type, and helps browsers to display web pages correctly.
    • It must only appear once, at the top of the page (before any HTML tags).
    • The < !DOCTYPE> declaration is not case sensitive.
    • The declaration for HTML5 is: < !DOCTYPE html>.
  2. < html>

      3. Syntax : < html>...< /html>

    • It is a double tag.
    • The < html> element is the root element of an HTML page.
  3. < head>

      4. Syntax : < head>...< /head>

    • It is a double tag.
    • The < head> element is a container for metadata (data about data) and is placed between the < html> tag and the < body> tag.
    • It also contain how and when a website will will show as a result by using name="" & content="" attributes.
    • Metadata is data about the HTML document. Metadata is not displayed.
    • Metadata typically define the document title, character set, styles, scripts, and other meta information.
  4. < title>

      5. Syntax : < title>...< /title>

    • It is a double tag.
    • The < title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab).
  5. < body>

      6. Syntax : < body>...< /body>

    • It is a double tag.
    • The < body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

Paragraph tag

Syntax : < p>< /p>

  • The < p> tag defines a paragraph.
  • Browsers automatically add a single blank line before and after each

    element.


Horizontal line tag

Syntax : < hr>

  • The < hr> element is used as a horizontal line that is used to separate content.

Breaking tag

Syntax : < br>

  • The < br> tag inserts a single line break.
  • The < br> tag is an empty tag which means that it has no end tag.
  • Note: Use the < br> tag to enter line breaks, not to add space between paragraphs.
